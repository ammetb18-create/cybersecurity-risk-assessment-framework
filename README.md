# Cybersecurity Risk Assessment Framework

## Overview

This project presents a cybersecurity risk assessment for a fictional financial data company. The analysis focuses on identifying critical assets, evaluating cybersecurity threats, estimating risk levels, and proposing risk treatment strategies.

## Objectives

- Identify critical information assets
- Evaluate cybersecurity threats
- Estimate risks based on probability and impact
- Develop risk mitigation strategies
- Support cybersecurity governance practices

## Methodology

This assessment follows cybersecurity risk management concepts aligned with:

- ISO/IEC 27005
- NIST Cybersecurity Framework
- CIA Triad (Confidentiality, Integrity, Availability)

## Key Threats Analyzed

- Ransomware
- Phishing attacks
- Unauthorized system access
- Cloud service failure
- Malware infection

## Deliverables

- Cybersecurity risk assessment report
- Asset valuation using CIA model
- 5x5 risk matrix
- Risk treatment plans
